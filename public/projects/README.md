# Project Images

Drop your 4 generated project images in this folder using these exact filenames:

| Project            | Filename                |
| ------------------ | ----------------------- |
| FileMind           | `filemind.png`          |
| Nandarani POS      | `nandarani-pos.png`     |
| Nandarani Catalog  | `nandarani-catalog.png` |
| AI Data Detective  | `ai-data-detective.png` |

Recommended size: **1280×800** (16:10) for project cards, square crops also work for the gallery.

Until you add them, the site shows a clean placeholder with the expected filename.

To change a filename or path, edit `lib/data.ts` → the `image` field of each project.
