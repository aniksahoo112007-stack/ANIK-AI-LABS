# Local Assets

Place images, videos, and PDFs here, then reference them from the admin panel
or lib/site-data.ts using a root-relative path (the `public/` prefix is dropped):

| File placed here                 | URL to use in admin / site-data |
| -------------------------------- | ------------------------------- |
| public/assets/filemind.png       | /assets/filemind.png            |
| public/assets/pos.png            | /assets/pos.png                 |
| public/assets/catalog.png        | /assets/catalog.png             |
| public/assets/demo-video.mp4     | /assets/demo-video.mp4          |
| public/assets/brand-book.pdf     | /assets/brand-book.pdf          |

These load instantly, work offline, and persist permanently because they ship
with the project. Use them for project images, gallery images, the video
showcase thumbnail/video, the Brand Book PDF, and the founder photo.
